# PPT Retouch

Minimal editor for final retouching of AI-generated HTML/CSS slides.

Architecture direction:

- [North Star](NORTH_STAR.md)
- [Standardization candidates](docs/standardization/README.md)

## Run

```sh
pnpm install
pnpm dev
```

## Check

```sh
pnpm lint
pnpm build
pnpm verify:mvp
```
